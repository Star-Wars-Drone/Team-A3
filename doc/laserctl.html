Usage:

zap = Laser(<optional_board_pin>,<optional_timeout_seconds>) #defaults to pin 12, 1 sec activation period
zap.trigger() #uses default timeout, or timeout specified in ctor
zap.trigger(1.5) #override (but don't set) timeout

Laser will deactivate (i.e. Pi output pin returns to low) after timeout period, OR if user calls

zap.switch_off() #deactivate laser and timer
